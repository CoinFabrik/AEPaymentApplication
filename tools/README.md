
Usage:

```bash

# create new account:

aecli account create  <account-file_name>     # default paassword is 1234

# transfer funds

NODE=10.10.0.79:3001 node tx.js  from  to  ammount

 - from: source wallet filename or "init" to use the base account
 - to: destination wallet filename
 - amount: amount to transfer

```
